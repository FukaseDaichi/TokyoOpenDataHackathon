## 3. もっとデータはあるか？ → 追加候補

都カタログは約54,000データセット（詳細: [../research/03-open-data.md](../research/03-open-data.md)）。
「区別に23区揃う×キャラの個性に翻訳できる」ものに絞った有望候補:

| 候補 | 何がわかる | キャラへの翻訳例 | 取得コスト |
|---|---|---|---|
| **地価公示（取得済み!）** | 区別の地価水準 | 「華やか」の補強、港・千代田・渋谷の格付け根拠 | 低（`chika_r7_chiten.csv` の区別集計のみ） |
| **警視庁犯罪オープンデータ** | 町丁字別×罪種別認知件数 | 「守り」軸・防犯意識の強いキャラ設定 | 低（登録不要CSV。ただしスティグマ回避の反転表現が必須） |
| **駅別乗降人員数** | 区内ターミナルの規模 | 「交通ハブ度」— 品川・新宿の通勤ハブ設定の実数根拠 | 低（都カタログCSV） |
| **外国人人口（住基台帳）** | 区別外国人比率 | 「国際派」設定（新宿・港・豊島が上位のはず） | 低（住基台帳の同シリーズ） |
| **令和7年国勢調査 速報人口** | 最新の人口・世帯数 | 人口規模＝キャラの「体格・存在感」に翻訳 | 低（2026-06公開済みCSV） |
| **事業所数・産業構成（経済センサス）** | ものづくり/商業/IT の産業色 | 墨田=職人、大田=町工場の実数根拠 | 中（e-Stat API、集計要） |
| **子ども食堂の分布** | 福祉・地域のつながり | 「人情」系キャラ（足立・葛飾）の根拠補強 | 低（座標付きCSV、ただし36自治体で欠損あり） |
| **緑のオープンデータGIS（2026新規）** | 街路樹・緑被率 | 「みどり」軸の精密化（公園以外の緑） | 中（中身未確認、要kill test） |
| **消費者物価指数（区部）** | 物価の時系列 | 区別には割れない → キャラ用途では優先度低 | — |

**判断の目安**: 現5軸で全23区が識別できることは確認済みなので、追加は「軸を増やす」より
**個別ページの根拠数値を厚くする**用途が費用対効果が高い。最有力は既に手元にある
**地価公示の区別集計**と、登録不要ですぐ落とせる**外国人人口・駅別乗降人員**の3つ。

いずれも採用時は品質ゲート（2時間kill test・スティグマ回避・欠損チェック）を通すこと。
