## 指標選定の判断メモ

- **みどり軸**は「公立公園合計（都立＋区立等）の一人当たり面積（ハ/Ｂ列）」を採用。
  国民公園（皇居外苑等）・公団公社を含む総合計（ホ/Ｂ）だと千代田区が25㎡/人に跳ね
  「皇居＝千代田区のみどり」という直感に反する歪みが出るため除外した。
- **賑わい軸**は千代田区1355%が支配的なため、正規化時はlogスケールを使う
  （z-スコア算出で確認済み。logでも千代田+3.4σで個性は保持される）。
- **華やか軸**は当初候補の地価公示（`data/raw/chika_r7_chiten.csv` 取得済み）より
  財政力指数を主指標に。地点データの集計が不要で、23区が単一表で揃うため。
  地価は個別ページの根拠数値の補強に使える。
- ダウンロード失敗ファイル（中身がHTML）: `estat_9-1-1.csv`, `metro_kt20tv0009.csv`。
  いずれも財政力指数の別経路だったため不要（soumu xlsxで代替済み）。
