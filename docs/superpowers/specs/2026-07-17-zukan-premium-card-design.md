# 図鑑一覧「プレミアムカード」リデザイン 設計書

日付: 2026-07-17
対象: トップページ「23区ちゃん図鑑」一覧（`src/ui/Zukan.tsx` / `app/zukan.css`）

## 目的

図鑑一覧カードを「収集したくなるソシャゲ召喚カード風」に刷新し、区ごとのテーマカラーと立ち絵を主役にする。現在表示している「系統」ラベル（k-meansクラスタ名）はカードから削除する。

スマートフォン閲覧が主である前提を置く。ホバー演出は加点要素とし、静的な見た目（色縁・立ち絵・金文字の区名）だけで魅力が成立するようにする。

## 変更しないもの（スコープ外）

- タップ/クリックでモーダルを開く挙動、`aria-label`、キーボード操作
- `zukanNo()` の採番ロジック（JIS区コード順）
- 画像の遅延読み込みと `is-loaded` フェードイン（Chromiumのlazy画像スキップ回避コメントごと維持）
- `src/data/wards.ts` の `group` 生成。系統は結果画面・シェアカード等で引き続き使用するため、削除するのは図鑑カード上の表示だけ
- 画像アセット。全23区の512px/896px WebPが揃っており新規生成は不要

## カード構成（案A）

### 静的レイヤー（モバイルでも常に見える本体）

1. **フルブリード立ち絵**: 2:3の立ち絵をカード全面に敷く。羊皮紙の余白・内枠は廃止
2. **テーマカラーの縁取り**: `--ward-color`（髪色由来）の2〜3pxボーダー。現在の一律金色 `#b8923f` から区ごとの色へ
3. **区名プレート**: カード下部に暗色グラデーション（透明→`rgba(23,17,12,0.85)`程度）を敷き、その上に区名を金〜クリーム色の文字で重ねる
4. **メダリオンNo.**: 左上に円形の金バッジ風でNo.を表示（現在のピル型バッジを格上げ）

### インタラクション演出（加点レイヤー）

- **ホバー環境**（`@media (hover: hover)`）: 浮き上がり＋`--ward-color`のグロー影＋シャイン（斜めの光沢が1回走る。`ward-modal-shine` と同系のkeyframesを図鑑用に用意）＋キャッチコピー（`wardTheme().catch`）が下からスライドイン
- **タッチ環境**: IntersectionObserverでカードが画面に入った瞬間にシャインを1回だけ再生する。キャッチコピーはカードに表示せず、タップ後のモーダルに任せる
- **`prefers-reduced-motion: reduce`**: シャイン・浮き上がり・スライドインをすべて無効化。静的レイヤーのみで成立させる
- **`:focus-visible`**: 現行同様 `--ward-color` のアウトラインを維持

### グリッド

- `minmax(148px, 1fr)` → `minmax(160px, 1fr)` に拡大し、スマホ2カラム時に立ち絵の顔がしっかり見えるようにする

## 実装上の注意

- `Zukan.tsx` から `zukan-card-group` 要素を削除し、カードDOMを「画像＋グラデーション＋区名＋No.＋（ホバー用）キャッチコピー」に組み替える
- シャインのDOM要素は装飾なので `aria-hidden="true"`
- IntersectionObserverはJS無効環境（`html.uk-js` が付かないSSR静的表示）では発火しない。その場合も静的レイヤーだけで成立するため問題ない
- 区名はグラデーション上の白抜き/金文字になるためコントラストを確認する（羊皮紙前提だった墨色 `#4a3418` は使えない）
- 既存の `zukan-card-placeholder`（slug=null時の下地）は分岐ごと残す。現在は全区slugありのため通常表示されない
