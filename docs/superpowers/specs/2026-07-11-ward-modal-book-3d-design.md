# WardModal「魔法の絵本」リニューアル 設計

日付: 2026-07-11
ステータス: 承認済み（ユーザーレビュー待ち）

## 目的

区ちゃん詳細モーダル（`src/ui/WardModal.tsx`）の体験を強化し、オーディエンス賞（バズ）向けの「ワクワクする」演出を実装する。

1. **開閉演出**: モーダルを「絵本の表紙が開く」アニメーションで開閉する
2. **3Dレーダー**: フラットSVGレーダーを three.js による「魔法陣ホログラム」3D表現に置き換える

## 決定事項（ブレインストーミングで確定）

| 論点 | 決定 |
|---|---|
| 3D実現方式 | three.js による本格WebGL 3D |
| 3Dビジュアルコンセプト | 魔法陣ホログラム（羊皮紙上に金の魔法陣＋上空に半透明の性格ポリゴン浮遊） |
| 操作性 | 自動回転＋ドラッグ/スワイプで手動回転（慣性付き、離すと自動回転に復帰） |
| 軸ラベル・順位 | 3D内ビルボードスプライト（「賑わい 3位」がポリゴンと一緒に回転、常にカメラ目線）。HTMLチップは3D有効時廃止 |
| 開閉演出 | 表紙が左端を軸に開く見開き（CSS 3D rotateY）。閉じるは逆再生 |
| 表紙デザイン | 共通表紙画像1枚＋区名・No.の金箔風テキストオーバーレイ |

## 制約（AGENTS.md 準拠）

- 静的エクスポート（`output: 'export'`）互換 — 全てクライアントサイドで完結
- 追加課金なし — three.js は npm の無料 OSS
- 既存の羊皮紙×金の絵本トーンを維持
- データ根拠の担保 — 順位・根拠数値は下段ステータス表を正として維持
- UIコピーは日本語

## 1. 3Dレーダー「魔法陣ホログラム」

### コンポーネント構成

- **`src/ui/Radar3D.tsx`**（新規、クライアント専用）: three.js を直接使用。react-three-fiber は導入しない（依存最小化）
- **`src/lib/radar3dGeometry.ts`**（新規、純ロジック・TDD対象）: AxisVector → 3D頂点座標列・目盛りリング座標・ラベル配置座標を返す副作用なし関数群。three.js 非依存の数値計算のみ
- 既存 `src/ui/Radar.tsx`（2D SVG）はフォールバックとして保持

### シーン構成（下から上へ）

1. **魔法陣**: 床に寝かせた円盤（透過PNGテクスチャ）。ゆっくり自動回転＋発光の脈動
2. **目盛りリング**: -1 / 0 / +1 の五角形ワイヤーフレームが薄い金線で浮遊
3. **性格ポリゴン**: 5頂点の半透明ゴールドメッシュ。わずかな厚み、縁は明るいワイヤー、頂点に光球。登場時は魔法陣から立ち昇るように浮上＋スケールイン（召喚演出）
4. **粒子**: 頂点から金の光の粒が立ち昇る（加算合成 Points）
5. **軸ラベル**: 「賑わい 3位」を CanvasTexture のスプライトで各頂点外側に配置。ビルボード（常にカメラ目線）

### カメラ・操作

- 斜め上約30°の見下ろしカメラ
- グループ全体が自動回転（約12秒/周）
- pointerdown〜pointermove でドラッグ回転（慣性付き）、離すと自動回転に復帰
- canvas は透過背景で `.ward-modal-radar` 羊皮紙パネルに重ねる

## 2. 絵本開閉演出

### DOM構造

```
.ward-modal（背景減光＋粒子: 既存）
└ .ward-modal-book（perspective: 2000px）
   ├ .ward-modal-cover（表紙。transform-origin: left、カードと同サイズ）
   │   ├ 表紙画像（共通1枚）＋ 区名・No. 金箔風オーバーレイ
   │   └ 裏面（見返し: 旧紙色の無地）
   └ .ward-modal-card（既存カード。表紙の下に待機）
```

### アニメーションシーケンス

- **開く（計約1.0秒）**:
  1. 閉じた本がふわっと中央に登場（0.25s）
  2. 表紙が `rotateY(0 → -105°)` で開く（0.55s）
  3. 90°を過ぎた瞬間に金の光フラッシュ＋表紙フェードアウト、カードが `scale(0.96→1)` で現れる（0.3s）
  4. 中身はヘッダー→レーダー召喚→ステータス行の順に stagger でフェードイン
- **閉じる**: `closing` 状態を導入し逆再生（表紙が戻ってパタン→全体フェード）。アニメーション完了後に `onClose` を呼ぶ。×ボタン・背景クリック・Escape すべて同経路

## 3. フォールバック・アクセシビリティ

- **`prefers-reduced-motion`**: 開閉は単純フェード、レーダーは既存2D SVG＋HTMLチップ（現行レイアウト温存）
- **WebGL不可 / three.js 読込失敗**: 2Dフォールバック
- **遅延読込**: three.js（gzip 約150KB）は `next/dynamic`（`ssr: false`）でモーダル初回表示時のみロード。ロード中は2Dレーダー→3Dへシームレス切替
- canvas に現行SVGと同等の `aria-label`（各軸数値の読み上げ）

## 4. 生成アセット仕様（画像生成で用意）

| アセット | 仕様 | 用途 |
|---|---|---|
| 表紙画像 1枚 | 横長 約1600×1200。革装風・金の縁飾り・タイトル「うちの区ちゃん図鑑」・区名オーバーレイ用余白 | 開閉演出の表紙 |
| 魔法陣テクスチャ 1枚 | 1024×1024 透過PNG。金の線画の魔法陣（円環＋幾何学模様）、背景完全透明 | 3Dレーダーの床 |

アセット到着まで CSS/SVG の仮表紙・プロシージャル仮魔法陣で実装を進め、画像差し替えのみで完成する構造にする。
